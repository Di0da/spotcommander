UPGRADE NOTES:

 * Close SpotCommander completely on all your devices before upgrading
 * Android users: Make sure you are running v 3.2 or newer of the Android app
 * iOS users: You should re-add to home screen
 * HTML5 application cache is no longer used due to a bug in iOS 7
 * Make sure you reload/refresh the web app at least twice on every device after upgrading
 * If stuck, you should clear the application cache: http://www.olejon.net/code/spotcommander/?clear_application_cache

CHANGELOG:

 * Support for v 3.2 of the Android app
 * On Android, flip to pause and persistent notification with playback controls
 * Support for Android 4.4
 * Support for iOS 7, including new home screen icon
 * Play starred tracks as playlist
 * Scroll to top by tapping the top action bar
 * Removed support for old versions of Opera
 * Minor changes
 * Bug fixes

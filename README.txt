UPGRADE NOTES:

 * Close SpotCommander completely on all your devices before upgrading
 * Android users: Make sure you are running v 3.2 or newer of the Android app
 * iOS users: If you are upgrading from v 8.8 or older, you should re-add to home screen
 * If you are upgrading from v 8.8 or older, make sure you reload/refresh the web app at least twice on every device
 * If stuck, you should clear the application cache: http://www.olejon.net/code/spotcommander/?clear_application_cache

CHANGELOG:

 * UI changes
 * Better artist and playlist cover art
 * Notifications (can be turned on in Settings if supported by the browser)
 * Switched to the new Spotify web API from the now deprecated metadata API
 * Improved queue daemon
 * Fixed a bug where adjusting the system volume did not work
 * Fixed lyrics when a track has multiple artists
 * Removed Ubuntu Unity integration since it no longer works on Ubuntu 14.04
 * Minor changes and bug fixes

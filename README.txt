UPGRADE NOTES:

 * Close SpotCommander completely on all your devices before upgrading
 * Android users: Make sure you are running Android app v 2.3 or newer
 * iOS users: Re-add to home screen if you haven't done so since v 8.3 or older
 * If some of your starred items have weird names after upgrading from v 8.4 or older, delete and star them again
 * If you browser gets stuck on "Updating cache..." after upgrading, you should clear the application cache:
   http://www.olejon.net/code/spotcommander/?clear_application_cache

CHANGELOG:

 * Toggle between adjusting Spotify's volume and the system volume
 * If PulseAudio is not installed, fall back to adjusting system volume through ALSA
 * Start radio fix in settings (if the up keyboard button is simulated wrong number of times)
 * When starting track radios, simulate two left-clicks to disable screensaver and hide advertisement
 * Highlight currently playing track in lists
 * Added new Spotify regions in settings
 * Due to problems in some browsers, only the checkboxes themselves can now be used to toggle settings
 * Support for v 2.3 of the Android app (pause on incoming call feature)
 * Authentication support (see wiki)
 * Changes & optimizations under the hood
 * Bug fixes
